# Gerenciador de Lojas em Shopping Center

O Gerenciador de Lojas em Shopping Center é um sistema desenvolvido para auxiliar no gerenciamento e controle das lojas localizadas em um shopping. 
O sistema permite o acompanhamento do espaço disponível para locação, a alocação de lojas em espaços específicos, a atualização das informações das lojas e a geração de relatórios.

Em um shopping, as lojas podem ser de diversos segmentos, como vestuário, alimentação, lazer, cinema, serviços, entre outros.

Objetivos
O objetivo deste projeto é criar uma aplicação que permita:

Gerenciar os espaços disponíveis e alugados no shopping.
Registrar e atualizar informações das lojas, como nome, segmento, e status de locação.
Gerar relatórios sobre as lojas e os espaços do shopping.
Permitir a locação e desocupação de espaços dentro do shopping.
Este sistema visa fornecer uma interface fácil de usar para administradores de shoppings, ajudando-os a gerenciar a alocação das lojas de forma eficiente e organizada.
